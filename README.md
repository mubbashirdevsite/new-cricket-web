# Professional Cricket Club

A modern and responsive cricket club website built with HTML5, CSS3, and JavaScript.

## Features

- Modern Glassmorphism UI
- Responsive Design
- Fixed Navigation Bar
- Hero Section
- About Section
- Featured Players
- Upcoming Matches
- Gallery
- Achievements
- Join Club Form
- Sponsors Section
- Contact Section
- Google Maps Integration
